수업 기록 제출 방법

매월 아래 순서대로 기록을 입력하고 관리자에게 제출합니다.

1

강사 모드 선택

접속 화면에서 초록색 강사 모드 버튼을 클릭합니다.

✏️ 강사 모드
수업 기록을
제출합니다
🔒 관리자 모드
정산 내역
관리합니다
2

이름 입력 및 로그인

관리자에게 등록된 이름과 비밀번호를 정확히 입력합니다.

⚠️ 이름은 공백·오탈자 하나도 없이 완전히 일치해야 합니다.
3

수업기록표 자동 로드 NEW

로그인하면 센터 DB에 등록된 스케줄이 자동으로 채워집니다. 별도로 불러오기 버튼을 누를 필요가 없습니다.

↑ 로그인 직후 스케줄이 자동으로 채워진 수업기록표

🆕
센터 셀 배경색 구분 — 센터명이 선택된 행은 연한 초록색 배경으로 강조됩니다. 빈 행은 흰색으로 표시되어 한눈에 구분됩니다.
💡 스케줄이 미등록된 강사는 월~금 빈 행으로 시작합니다. 관리자에게 문의하세요.
4

출결 입력

각 날짜 셀을 클릭할 때마다 기호가 바뀝니다.

O
정상 출석
1회 카운트
X
결석
0회
△
보강
0.5회

빈 칸 → O → X → △ → 빈 칸 순으로 순환합니다.

5

특이사항·영수증 입력

신규 수업, 휴강, 보강, 주차료 등 특이사항을 태그로 추가합니다. 없으면 건너뛰세요.

+ 신규(모의) 수업 + 휴강 + 보강 + 주차료 + 기타
💡 계좌번호는 로그인 이름 기준으로 자동 채워집니다.
6

정산 미리보기 확인

입력 즉시 자동 계산됩니다. 세전 강사료와 확정 월급을 확인하세요.

강사료 유형
고정제
세전
840,000
확정 월급
812,280
ℹ️ 확정 월급 = 세전 × (1 − 3.3%). 고정제는 회당 금액 × 출석 횟수.
7

임시저장 또는 제출

작성 중이면 임시저장, 완성됐으면 관리자에게 제출을 누릅니다.

💾 임시저장
✉️ 관리자에게 제출
📥 제출 전 엑셀 다운로드로 기록을 저장해둘 수 있습니다.
