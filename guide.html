시스템 개요

두 가지 모드로 운영됩니다

강사는 수업 기록을 직접 입력해 제출하고, 관리자는 정산을 처리합니다.

강사 모드

강사용

수업 기록을 입력하고 관리자에게 제출합니다

  • 센터별 수업 횟수 입력
  • 정산 미리보기 확인
  • 엑셀 다운로드
  • 관리자에게 제출
관리자 모드

  • 월별 정산 조회
  • 강사 등록·수정
  • 센터 관리
  • 성장추이 분석
💾
브라우저 자동 저장 — 모든 데이터는 브라우저 로컬 저장소에 자동 저장됩니다. 단, 다른 기기에서는 별도로 관리됩니다.
강사 모드

수업 기록 제출 방법

매월 아래 순서대로 기록을 입력하고 관리자에게 제출합니다.

1

강사 모드 선택

접속 화면에서 초록색 강사 모드 버튼을 클릭합니다.

✏️ 강사 모드
수업 기록을
제출합니다
🔒 관리자 모드
정산 내역
관리합니다
2

강사 이름 입력

등록된 이름을 정확히 입력합니다. 공백·오탈자 하나도 오류 원인이 됩니다.

⚠️ 관리자에게 등록된 이름과 완전히 일치해야 합니다.
3

연도·월 선택

기록할 수업의 연도와 월을 선택합니다. (기본값: 현재 월)

4

수업 기록표 입력

요일별로 센터를 등록하고 매주 출결을 기록합니다. 셀을 클릭할 때마다 기호가 바뀝니다.

○
정상 수업
1회 카운트
✕
휴강
0회 (미진행)
△
보강·신규
0.5회
💡 날짜 자동채우기 버튼을 누르면 해당 월의 날짜가 자동 입력됩니다.
5

특이사항·계좌·영수증 입력

특이사항이 있거나 영수증을 첨부해야 할 경우 입력합니다. 없으면 건너뛰어도 됩니다.

+ 신규(모의) 수업 + 휴강 + 보강 + 주차료 + 기타
💡 계좌번호는 강사 이름 입력 시 자동 채워집니다.
6

정산 미리보기 확인

수업 기록 입력 시 자동 계산됩니다. 내용을 확인하세요.

강사료 기준
80%
세전
2,156,000
확정 월급
2,084,852
ℹ️ 비율제는 수업료 합계 × 요율, 고정제는 횟수 × 회당 금액. 확정 월급은 세전에서 3.3% 공제 후.
7

임시저장 또는 제출

아직 작성 중이면 임시저장, 완성됐으면 관리자에게 제출을 누릅니다.

💾 임시저장

강사 모드 선택 → 이름 입력 → 연도·월 → 수업 기록 → 특이사항 → 미리보기 → 제출
관리자 모드

정산 및 관리 방법

관리자 비밀번호로 로그인 후 네 가지 탭으로 업무를 처리합니다.

🔒
관리자 로그인 — 모드 선택 화면에서 관리자 모드 클릭 → 비밀번호 입력 → 로그인. 로그인 즉시 인상예정일이 지난 강사료가 자동 적용됩니다.
🌿 수업기록표 📊 정산 👥 강사 🏢 센터 📈 성장 🔔 알림
📊

정산 탭

월별 수업기록 조회 및 강사별 정산
①
강사 제출 데이터 반영

강사가 제출하면 상단에 초록 배너가 표시됩니다. 반영하기 버튼을 누르면 정산 화면에 즉시 적용됩니다.

📬 3명의 수업기록표가 대기 중이에요
반영하기
②
월별 조회

좌측 사이드바에서 조회할 월을 선택합니다. 저장된 월 목록이 최신순으로 표시됩니다.

③
강사별 정산 상세

강사 이름 클릭 → 수업료 합계 · 세전 강사료 · 3.3% 세금 · 확정 월급 · 최종 지급액 확인.

④
지역 필터 & 다운로드

서울 / 인천 / 수원으로 필터링하거나, CSV·PDF 버튼으로 정산표를 저장할 수 있습니다.

👥

강사 관리 탭

강사 등록·수정·삭제 및 강사료 관리
①
강사 목록 조회

지역별 강사 수와 비율/고정 유형 통계 확인. 지역 탭 또는 이름으로 필터링 가능합니다.

②
강사 정보 수정

강사 행 클릭 → 수정 모달. 이름·지역·강사료 유형·회당 금액·계좌번호·입사일·인상예정일 수정 가능.

⚡
입사일 입력 시 인상예정일(입사 6개월 후)이 자동 계산됩니다.
③
신규 강사 등록

우측 상단 + 강사 추가 버튼으로 등록합니다. 이름은 강사 모드 로그인 이름과 동일하게 입력해야 합니다.

🏢

센터 관리 탭

269개 센터 정보 및 수업료 관리
①
센터 조회 & 수정

센터명 검색, 지역·강사 기준 필터링 가능. 행 클릭 시 센터명·지역·기본 수업료·담당 강사 수정 가능.

②
마스터 수업료 규칙

센터에 기본 수업료가 등록되어 있으면 강사 입력값보다 우선 적용됩니다. 강사별로 다른 수업료라면 비워두세요.

📈

성장추이 탭

월별 통계 및 인상 예정 현황
①
전체 현황 요약

기록된 총 월 수, 강사 수, 센터 수, 최근 저장 일자를 한눈에 확인합니다.

②
월별 변화 추이

지난달 대비 변경사항 · 두 달 직접 비교 · 올해 전체 데이터 3가지 뷰로 확인 가능합니다.

③
강사료 인상 예정 현황

인상예정일 목록과 D-n 배지 표시. 인상예정일 도래 시 다음 로그인 때 자동 42,000원 인상 적용.

🔔

알림 탭

계좌 변경 알림 및 업데이트 내역
①
계좌번호 변경 대기

강사가 수업 제출 시 계좌를 변경하면 파란 배너와 알림 탭에 표시됩니다. 반영하기로 DB에 즉시 적용.

💳 2명의 계좌번호가 변경됐어요
반영하기
②
업데이트 내역

시스템 버전별 변경사항을 확인할 수 있습니다. 현재 버전은 상단바 로고 옆에 표시됩니다.
